# ConsolidatedUI aka HXUI
(BETA) Ashita v4 addon containing multiple ui elements:

1) Player Bar
2) Target Bar
3) Party List
4) Enemy List
5) Exp Bar
6) Inventory Tracker
7) Gil Tracker
8) Configuration UI

Configuration UI Command: /cui, /consolidatedui, /hxui, /hui, /horixonui, /horizonxiui
